## ⚙️ Configuración para Compilar
Este proyecto requiere librerías externas en la unidad C:/:
1. **OpenCV Android SDK:** Extraer en `C:/OpenCV`
2. **ONNX Runtime:** Extraer en `C:/ONNX` (con carpetas `headers` y `jni`)

Si tus rutas son diferentes, modifica el archivo `app/build.gradle.kts` y `CMakeLists.txt`.
